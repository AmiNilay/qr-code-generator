# QR Code Generator

A sleek and modern QR Code Generator built with HTML, CSS, and JavaScript. 

[**View Live**](https://aminilay.github.io/qr-code-generator/)

---

## 🌟 Overview

This QR Code Generator provides a fast, user-friendly interface to create custom QR codes for a wide variety of data types. It features a modern, tech-inspired design with a clean purple aesthetic and displays live usage statistics.

## ✨ Features

* **Multiple Data Types:** Easily switch between generating QR codes for URLs, Text, Email, Wi-Fi, Phone numbers, SMS, vCards, PDFs, Apps, and Multi-URLs.
* **Live Statistics Dashboard:** Displays real-time metrics including "Users Online," "Generated Today," and "Total Generated."
* **Modern UI:** A clean, responsive layout with a dedicated, clean preview panel for your generated QR codes.
* **Instant Generation:** Fast processing to get your QR codes immediately.

## 🛠️ Tech Stack

* **HTML5**
* **CSS3** (Modern layout, gradients, and styling)
* **Vanilla JavaScript** (Core logic and QR code generation)

## 🚀 Getting Started

To run this project locally:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/aminilay/qr-code-generator.git](https://github.com/aminilay/qr-code-generator.git)
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd qr-code-generator
    ```
3.  **Open the site:**
    Simply open the `index.html` file in your preferred web browser to start generating QR codes.
